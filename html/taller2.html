Evento Descripción Ejemplo

OnClick

El evento "click" se dispara cuando se presiona el botón de un dispositivo de entrada (Mouse o Mousepad) sobre un elemento.
Blur

El evento blur es disparado cuando un elemento ha perdido su foco. La diferencia principal entre este evento y focusout (en-US) es que sólo el último se propaga (bubbles).
Click

El evento "click" se dispara cuando se presiona el botón de un dispositivo de entrada (Mouse o Mousepad) sobre un elemento.

OnLoad

Se dispara cuando la página HTML y todos sus recursos se han cargado completamente.

Es el alert que aparecio al cargar esta página
OnKeydown

Se dispara cuando se presiona una tecla en un campo de entrada.
MouseEnter

Se activa cuando el puntero del mouse entra en un elemento. Es similar al evento "mouseover", pero no se propaga a los elementos secundarios.
MouseLeave

Se dispara cuando el puntero del mouse sale de un elemento. Al igual que "mouseenter", no se propaga a los elementos secundarios
Focus

Se activa cuando un elemento obtiene el foco. Puedes usarlo para resaltar o mostrar información adicional sobre un elemento cuando el usuario interactúa con él.
Sumbit

Se dispara cuando se envía un formulario. Puedes usarlo para validar los datos antes de enviarlos al servidor.
DblClick

Este evento se utiliza comúnmente para capturar acciones del usuario que involucran doble clic, como abrir una ventana emergente o ampliar una imagen.
